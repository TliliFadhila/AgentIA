# EUROPROD Smart Infrastructure Advisor - CSV Datasets

1. questions_master + answers_master = UX conversationnelle exacte
2. answer_impacts_rules = moteur d'inference simple
3. regulations_master + regulation_requirements = logique reglementaire
4. technologies_master + catalogue_items_dictionary = dictionnaire technique/catalogue
5. segment_baseline_matrix + stack_rules = recommandation par segment et architecture
6. report_output_schema + priority_dictionary = generation du rapport client

Tous les identifiants et labels sont normalises ASCII pour eviter les caracteres problematiques en ingestion modele/API.
Les textes client peuvent etre reaccentues au niveau front-end si necessaire.
